# Getting Started
* Building: mvn clean package or ./mvnw clean package
* Running: java -jar target/spring-boot-project-template-1.0.0.jar or using IDE Run/Debug
* Web UI: http://localhost:8080
* REST API: http://localhost:8080/api/reservations?date=2020-01-01
